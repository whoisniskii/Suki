# Suki-Rewrite

## ⚠ Requirements
- NodeJS 16 or higher
- yarn


## 💻 Self-Hosting

- ⚙️ Rename `src/Config/config.example.ts` to `config.ts` and fill out the values.

- Install all dependencies
```bash
yarn
```

## 💻 | Compiling
To compile the bot run:

```bash
yarn build
```

Now you can run bot using:
```bash
yarn start
```
